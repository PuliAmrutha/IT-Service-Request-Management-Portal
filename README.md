# IT Service Request Management Portal

## Project Overview

The IT Service Request Management Portal is a custom ServiceNow application developed to simplify IT service request management within an organization.

Employees can submit IT service requests, technicians can manage assigned requests, managers can monitor progress, and administrators have full control over the application.

---

## Features

- Create IT Service Requests
- Assign Requests to Technicians
- Request Status Tracking
- Email Notifications
- Flow Designer Automation
- Business Rules
- Client Scripts
- UI Policies
- Access Control Lists (ACLs)
- Reports and Dashboards
- Role-Based Access Control

## Roles

### Employee
- Create Service Requests
- View Own Requests

### Technician
- Update Assigned Requests

### Manager
- Monitor Service Requests

### Admin
- Full Application Access

---

## Workflow

1. Employee submits a service request.
2. Request is assigned to a technician.
3. Technician updates the request.
4. Email notifications are sent.
5. Reports and dashboards display request statistics.
